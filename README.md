# idle-waygo-inhibitor

Simple idle inhibitor per Wayland in Go.
It was totaly generated by Claude.

## Install

```bash
go build
go install
```

## Use

Use `-n` option to send notification of the state via `notify-send`.

```bash
./idle-waygo-inhibitor [-n] <start|stop|toggle|status>
```
